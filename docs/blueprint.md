# **App Name**: ProcrastinatorPortal

## Core Features:

- User Authentication & Humorous Login Flow: Secure user registration with specific password criteria and humorous validation, followed by an intentionally frustrating login experience featuring duplicating buttons, fake loading, and a deceptive CAPTCHA challenge before access.
- Dynamic ProcrastiBot AI Assistant: An AI-powered chatbot offering context-aware sarcastic comments, an excuse generation tool for tasks, a bad suggestion generation tool for counter-productive distractions, and providing humorous interpretations of analytics or badge lore. It can also generate 'professional excuse' emails.
- Task Delay Kanban Board: A satirical Kanban board with columns like 'Maybe Later' and 'Not In This Lifetime', allowing users to drag and postpone tasks with humorous notifications. Features a 'Complete' button with a guilt-tripping modal and an AI tool to generate 'Distraction Tasks' based on user input.
- Chaos Calendar Event Manager: An event scheduling system where the calendar 'explodes' into draggable date blocks. Users must manually reassemble the calendar to set event dates, turning a simple action into a mini-game.
- Gamified Failure Achievements: A 'Achievement Scam' page displaying badges for unproductive behaviors, such as failed logins, excessive postponements, and chatting with ProcrastiBot, with animations and integrated lore from the AI.
- Satirical Overkill Analytics Display: A visually dense dashboard featuring absurd charts and graphs (e.g., 'Return on Inactivity', 'Login Suffering Index'), with micro-copy roasts and interactive elements that resist displaying positive data.
- Dynamic Notification Overload Center: A customizable notification panel that allows users to control the 'spam density' and 'tone' of purposefully useless, humorous notifications related to procrastination, with a live feed preview.

## Style Guidelines:

- Primary Color (general app elements): Soft lavender, `#C2B3E6`. This subtle shade provides a playful yet sophisticated base that resonates with the app's humorous intent without being overtly childish.
- Background Color (general app background): Extremely light lavender, `#F8F5FF`. Maintaining the same hue as the primary color, this very light, desaturated tone offers an airy backdrop for content, creating a clean and soft canvas for interaction.
- Accent Color (CTAs, highlights): Vibrant blue, `#2661D9`. Analogous to the primary, this energetic blue provides a distinct, high-contrast element for calls to action and important interactive components, ensuring visibility while upholding a playful vibe.
- Dashboard Palette: A separate dark theme utilizing deep charcoal backgrounds, neon violet accents, and glowing amber 'warning' highlights (`Nuclear Sunset` style) will be applied to the main dashboard interface to evoke an 'enterprise-grade' yet satirical feel.
- Headline Font: 'Poppins' (geometric sans-serif). Its precise and contemporary aesthetic pairs well with the app's modern, tongue-in-cheek personality for titles and short statements.
- Body Text Font: 'PT Sans' (humanist sans-serif). Chosen for its balanced blend of modernity and warmth, providing excellent readability for descriptive paragraphs and longer UI text, complementing the playful headlines.
- A diverse collection of playful and thematic icons, with a strong emphasis on animated elements, emoji-style visuals (especially for achievement badges), and 'procrastination-themed' illustrations, reinforcing the app's humorous tone.
- Clean and well-structured layouts with prominent 'rounded buttons' are central to the design. The UI prioritizes visual appeal and clear hierarchy to guide users, ensuring an engaging and easily navigable experience, despite the humorous complexity.
- Extensive use of smooth micro-animations for button interactions, content transitions, loading sequences, and dynamic visual elements like floating task cards, falling calendar blocks, and glowing badge states. These animations enhance humor and overall user engagement.
